Covalent

Covalent provides a unified API to bring full transparency and visibility to assets across all blockchains. The Covalent API can be used to build the {{ .Get "dex" }} analytics dashboard.

To get started, sign up for a Covalent API Key.

Developer Mode (JSON) Analyst Mode (CSV)

Build Analytics Dashboards

Here is a breakdown of the the specific API endpoints that can be used to build the {{ .Get "dex" }} analytics dashboard:

Marker Endpoint Data
1 /{{ .Get "apiReferenceChain" }}/address/:address/transactions_v2 Timeseries TVL based on decoded Deposit and Withdraw events
2 /{{ .Get "apiReferenceChain" }}/xy=k/{{ .Get "dex" | lower }}/pools/address/:address 24h volume as a timeseries chart
3 /{{ .Get "apiReferenceChain" }}/xy=k/{{ .Get "dex" | lower }}/tokens Top 50 tokens by price, volume, liquidity
4 /{{ .Get "apiReferenceChain" }}/xy=k/{{ .Get "dex" | lower }}/pools Top 50 pools by price, volume, liquidity
